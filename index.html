Hello hugo

Tue, May 14, 2019 Hello hugo win安装hugo： 到https://github.com/gohugoio/hugo/releases下载release版本的压缩包，解压后将hugo.exe的所在目录添加到环境变量。 测试hugo： 在cmd输入hugo vesion，如果安装成功，则有如下信息输出： Hugo Static Site Generator v0.55.5-A83256B9 windows/amd64 BuildDate: 2019-05-02T13:04:07Z 建立博客 本文以博主自己建立博客的代码为例，用到的 2hongBlog 和 zhong-yj 分别为博主的 博客本地文件夹名 和github username ，在建博客时应更换成自己相应的名字。 切换到你要建立博客的根目录下输入: hugo new site 2hongBlog 成功后当前目录下会出现一个2hongBlog的文件夹 ​ 2. 通过git init命令把这个目录变成Git可以管理的仓库： cd 2hongBlog git init ​ 3. 选择主题： 到https://themes.gohugo.io/选择一款自己喜欢的主题，这里以hyde这款主题为例，点进去里面有相应的git操作： cd themes git clone https://github.com/spf13/hyde.git 配置config.toml文件： 添加 theme = "hyde" 将baseURL更改为自己的域名: 自己新建一个github repo，注意这里的repo name格式必须是：github username.github.io
